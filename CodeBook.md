# 'Getting and Cleaning Data Course Project'

## 'The tidydata set'

The complete tidy dataset includes 68 columns and 180 rows.

## 'Data'

The data has been grouped by activity and subject, and then an average of each column has been calculated. 

## 'Columns'

The activity column is numbered 1 to 6. Each number relates to the following:
1.WALKING
2.WALKING_UPSTAIRS
3.WALKING_DOWNSTAIRS
4.SITTING
5.STANDING
6.LAYING
